1.Закидываем папку Agent в папку Документы

2. Настроить файл config.ini
    - name=Хлеб-Шувалова
    - update=1 - указать 0 если обновления не нужны
    - time=60 - время в минутах для проверки актуальности лог файла
2. Удалить ключ из файла ID.txt
3. Открыть редактор PowerShell от админа
4. Получить через бота короткий код, указав имя (имя точки) и группу (группа=название клиента)
5. Короткий полученный код поместить в файл code.txt
6. Открываем терминал от админа.
7. Пишем и запускаем: Set-ExecutionPolicy RemoteSigned Если скрипт находится уже в рабочем каталоге, запустить addplanner с правами администратора из редактора PowerShell
8. Убедится, что в планировщик добавилось задание
9. Запустить основной файл скрипта через редактор PowerShell
